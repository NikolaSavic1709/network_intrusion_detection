# Detekcija Napada na Mrežnih 

Projekat za detekciju mrežnih napada korišćenjem mašinskog učenja na NSL-KDD datasetu.

## Struktura

- `Network_Intrusion_Detection.ipynb` - Glavni py notebook - treniranje i evaluacija modela
- `eda.ipynb` - Eksplorativna analiza podataka (EDA)
- `requirements.txt` - dependencies

## Pokretanje

```bash
# Instalacija zavisnosti
pip install -r requirements.txt

# Pokrenite notebook fajlove u Jupyter
jupyter notebook
```
