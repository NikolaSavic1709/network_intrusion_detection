### 4. What The Notebook Does

The notebook executes the following pipeline:

1. **Data Loading** (~10 seconds)
   - Loads 25,193 training samples
   - Loads 22,543 test samples
   - Displays dataset statistics

2. **Preprocessing** (~30 seconds)
   - Encodes categorical features
   - Handles missing values
   - Normalizes features using StandardScaler

3. **Model Training** (~2-3 minutes)
   - Logistic Regression: ~2 seconds
   - Random Forest: ~30 seconds
   - SVM: ~45 seconds
   - Gradient Boosting: ~15 seconds
   - Neural Network: ~30 seconds

4. **Evaluation** (~1 minute)
   - Calculates accuracy, precision, recall, F1-score
   - Generates confusion matrices
   - Creates ROC curves
   - Generates visualizations

5. **Results Saving** (~10 seconds)
   - Saves best model to `src/models/`
   - Creates comparison CSV
   - Generates summary report

**Total Execution Time**: ~5-7 minutes

### 7. Troubleshooting

#### Issue: "Module not found" error
**Solution:**
```bash
pip install --upgrade pandas numpy scikit-learn matplotlib seaborn joblib
```

#### Issue: Notebook takes too long (>10 minutes)
**Solution:**
- Reduce training set size (use smaller subset)
- Reduce number of models
- Use machine with more RAM

#### Issue: "Cannot find data files"
**Solution:**
```bash
# Verify data files exist:
ls ../data/KDDTrain+_20Percent.txt
ls ../data/KDDTest+.txt

# If missing, check data folder path is correct relative to src/
```

#### Issue: Out of memory error
**Solution:**
- Close other applications
- Use full training set instead of 20% subset
- Reduce model complexity
